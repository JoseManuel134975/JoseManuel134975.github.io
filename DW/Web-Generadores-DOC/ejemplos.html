Ejemplos

Doxygen

Uno de los generadores más populares, soporta numerosos lenguajes (C++, C, Java, Python, etc.) y genera documentación en diversos formatos (HTML, LaTeX, etc.).

MKDocs

(Python) Sencillo y rápido, perfecto para proyectos de código abierto. Utiliza Markdown para escribir la documentación y genera sitios web estáticos.

JSDoc

Especializado en documentar código JavaScript. Utiliza comentarios similares a Javadoc para generar documentación HTML.
